CompanionEventHandler

open class CompanionEventHandler

Allows for an easy way to handle those pesky HandlerLists in Bukkit.

By extending this event you'll gain the wonderful handlerMap I've created. All that's left for you will be overriding the getHandlerList() method, marking it as @JvmStatic, and calling super.

companion object : CompanionEventHandler() {
     @JvmStatic
     override fun getHandlerList() = super.getHandlerList()
}
