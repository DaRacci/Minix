@API(status = API.Status.STABLE, since = "3.2.0")
enum LoggingLevel : Enum<MinixLogger.LoggingLevel>
Content copied to clipboard

A Utility for using levels between common loggers. LoggingLevel's are sorted by high level to low-level logs. This allows for easy filtering using Int comparisons.
